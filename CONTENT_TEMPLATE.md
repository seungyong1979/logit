# Logit 콘텐츠 작성 템플릿

> AI 초안 → 사람 편집 → 검수 → 발행 플로우에 최적화된 글 템플릿입니다.
> 이 템플릿을 ChatGPT/Claude에 붙여넣고 주제를 입력하면 초안을 생성할 수 있습니다.

---

## 📋 글 작성 전 체크

- [ ] 직접 사용해봤는가? (최소 2주 이상)

- [ ] 독자가 이 글을 읽고 얻을 수 있는 것은?
- [ ] 유사한 글이 이미 있다면 차별점은?

## 🤖 AI 초안 생성 프롬프트 (ChatGPT/Claude용)

```
아래 조건으로 블로그 글 초안을 써줘:

주제: [글 주제]
대상 독자: [대상 독자]
목표 키워드: [주 키워드], [보조 키워드 1], [보조 키워드 2]
글의 목적: [정보 제공 / 비교 / 후기 / 가이드 / 등]
내가 직접 경험한 것: [내 경험 요약 — AI가 대체 불가한 부분]
글 분량: 약 [1500~2500]자
말투: 전문적이지만 딱딱하지 않게, 1인칭 경험 기반

구조:
1. 공감 가는 도입부
2. [소제목 1]
3. [소제목 2]
4. [소제목 3]
5. 체크리스트 또는 표
6. FAQ 2~3개
7. 결론

중요: 수치나 통계는 "[확인 필요]" 표시를 붙여줘. 내가 직접 검증할 예정.
```

## ✏️ 사람 편집 체크리스트 (AI 초안 → 발행 버전)

### 필수 추가 사항
- [ ] 내 직접 경험 사례 최소 1개 추가
- [ ] AI가 "[확인 필요]" 표시한 수치 모두 검증
- [ ] 너무 일반적인 내용은 구체화 ("이 도구는 좋습니다" → "3개월 사용 후 ~부분에서 특히 도움이 됐습니다")
- [ ] "또한", "~를 통해", "~가 중요합니다" 과도한 반복 표현 다듬기
- [ ] 나만 알고 있는 팁이나 주의사항 추가
- [ ] 실패했거나 기대와 달랐던 경험 포함 (과도한 긍정 방지)

### 톤 확인
- [ ] 너무 홍보성 느낌이 나지 않는가?
- [ ] 내 목소리가 살아있는가?
- [ ] 독자에게 실제로 도움이 되는 정보인가?
- [ ] 검색 엔진이 아닌 사람을 위해 쓴 글인가?

### 최종 발행 전
- [ ] 모바일에서 가독성 확인
- [ ] 맞춤법/띄어쓰기 검사
- [ ] 이미지/썸네일 준비
- [ ] 관련 글 링크 삽입
- [ ] 발행 날짜 및 예상 읽기 시간 기재

## 📚 글 유형별 구조 가이드

### 도구 비교 글 (A vs B)
- 도입: 어떤 상황에서 이 고민을 했나
- 비교 기준 설정 (내가 중요하게 본 항목들)
- 기준별 비교 표
- 각 도구의 강점/약점
- 어떤 사람에게 뭐가 맞는지
- 결론: 내 선택과 이유

### 사용 후기/리뷰 글
- 도입: 왜 이 도구를 쓰게 됐나
- 기대했던 것과 실제
- 좋았던 점 (구체적 사례)
- 아쉬운 점 (솔직하게)
- 누구에게 추천하고 누구에게는 비추천인지
- 현재도 계속 쓰는지 여부

### 방법론/가이드 글
- 도입: 왜 이 방법이 필요한가
- 개요/원칙 설명
- 단계별 실전 가이드 (Step 1, 2, 3...)
- 흔한 실수와 주의사항
- 체크리스트
- 결론: 시작하는 방법

### 수익화/실험 기록 글
- 도입: 어떤 실험을 왜 했나
- 실험 전 조건 (시작 상태)
- 과정 (무엇을 어떻게 했나)
- 결과 (수치 포함)
- 배운 것과 다음에 할 것
- 독자에게 적용 가능한 인사이트
